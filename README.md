Este projeto foi desenvolvido para a disciplina de **Programação Orientada a Objetos (2025/01)**. O sistema processa dados da Justiça Eleitoral referentes à votação de vereadores em municípios brasileiros nas eleições de 6 de outubro de 2024.

O arquivo `especificacao.pdf` explica o escopo do trabalho. O projeto foi solucionado com duas implementações, uma em C++ e outra em Java.

> **Recomendação:** Recomendo que comece pela implementação em Java. Nela contém o diagrama UML e um relatório contendo uma explicação de como as classes são organizadas e se associam.
